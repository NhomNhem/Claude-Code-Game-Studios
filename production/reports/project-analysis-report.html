4. Cross-GDD Review (2026-05-26)

Verdict: FAIL — 3 Blocking Issues

🔴
R1: Time.timeScale authority conflict — GState vs Time System
Game State Manager AC: "unpause restores previousTimeScale (0.5)". Time System AC: "transitions from Paused to InRun → Time.timeScale == 1". Hai AC đối lập cho cùng một operation. Phải resolve trước khi implement.
🔴
R2: Event Bus missing Victory in Save subscription
Event Bus chỉ liệt kê Save System subscribe GameStateChangedEvent(HeroCamp, Defeat). Save/Profile GDD liệt kê Victory là 1 trong 4 autosave triggers. Không có subscription path cho Victory.
⚠️
W1: Formula variable startingGold undefined
save-profile-persistence.md dùng startingGold trong fraud check nhưng không có definition. Currency System (#16, vừa được tạo) cần define.
⚠️
W3: GPM/PPH/PDR constants are stubs
Fraud detection formulas dùng placeholder values (GPM 100-500, PPH 1-3, PDR 1.0-1.5) từ Currency System chưa started. Cần reconcile trước implementation.
⚠️
W5: HeroCamp UI navigation thiếu keyboard/gamepad
Input System routing table để NavigateUI, Confirm, Back = ✗ cho HeroCamp. Player bắt buộc dùng mouse trong camp — vi phạm P3 (Snappy Sessions).
⚠️
W6: BossActive flag ownership undefined
GState có open question "Who owns BossFight flag?" với 3 options, unresolved. Event Bus liệt kê interaction với Wave Spawning nhưng downstream systems assume flag exists.
ℹ️
W4: Entity registry empty
entities.yaml tất cả sections rỗng. Mọi cross-reference phải validate thủ công qua full GDD read. Chưa có conflict baseline.
